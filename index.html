Options that can be added to query string for garden.html. Later these will be added to some options widget, but for now, they can be specified this way. Boolean values can be given as 0|1|true|false.
    muse=bool
    Run MUSE message server (using socket.io)
    faces=bool
    Show faces on flowers
    lookAtH=height
    Make faces look towards point given height above mouse. height=200 is good value
    hud=bool
    Make a display window that hovers over garden and doesn't move as garden is panned
